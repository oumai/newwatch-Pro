• When set up the “Emergency Number” and “Family Number” in the app, why didn’t it sync to the smart watch immediately?

    In order to save battery, data of the watch will be uploaded to the server every hour. If you want to upload your data immediately after updating your details, please shut down the smart watch for 5 minutes and switch it on again. The number will upload immediately.

  • Why when pressed the “Emergency button”, the positioning information is not in the map?

    The information will be updated when the GPS, GSM position data is successfully delivered to the server, and according to the different positioning type, it will show different result and accuracy.

  • Why enable 'fall detection' function, sometimes it still a warning when you not fall down？

    Fall detection as an auxiliary function，This function is for reference only，Users can decide whether to enable。when Fall，you can press the red button for help.

  • How much data will the smart watch consume?

    Normally for standby without any SOS, the minimum monthly data usage is about 2-3Mbs. Usage is still based on the actual usage and frequency of use.

  • If set up more than one “Emergency number”, how does it work?

    This will lead in to a loop, whereby if the first number in the list didn’t pick up the phone, then it will automatically dial the next number in the list. Note that this doesn’t work with voice mail.

  • What’s the difference with “Emergency Number” and “Family Number”?

    The “Emergency Number” when pressed will dial and do a GPS positioning immediately. The “Family Number” when pressed will dial the number immediately but does not support real time positioning function.

  • How to set up the smart watch’s Bluetooth measurement equipment?

    In the setup screen, choose “Wireless Settings>Bluetooth”. Power on the Bluetooth and select “My device” and select the equipment that you would like to pair. Tap on and switch on your equipment, you should be able to find it and pair together.
